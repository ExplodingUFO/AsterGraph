# AsterGraph.Starter.Wpf

`AsterGraph.Starter.Wpf` is a validation-only adapter-2 sample for the WPF shell.

It exists to prove that the WPF adapter can compose the shared editor/session owner and platform
seams on Windows. It is not the default onboarding route, not a starter template for new hosts, and
not an Avalonia/WPF parity promise.

For new hosted UI work, start with `AsterGraph.Starter.Avalonia`, `dotnet new astergraph-avalonia`,
or the `AsterGraphHostBuilder` Avalonia route. Use this sample only when validating the WPF adapter
boundary described by the adapter capability matrix.
