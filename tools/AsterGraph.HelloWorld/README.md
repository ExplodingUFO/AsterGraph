# AsterGraph.HelloWorld

Smallest runtime-only first-run sample for AsterGraph.

This project proves only the runtime-only route and sits after `AsterGraph.Starter.Avalonia` in the public onboarding path:

- `AsterGraphEditorFactory.CreateSession(...)`

Run it with:

```powershell
dotnet run --project tools/AsterGraph.HelloWorld/AsterGraph.HelloWorld.csproj --nologo
```

Use this sample when you want the quickest adoption starting point for custom UI or native shells. Use `tools/AsterGraph.HostSample` when you need proof-oriented validation for the canonical runtime-only and hosted-UI routes.
