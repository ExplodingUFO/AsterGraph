# 交互与快捷键

AsterGraph 默认带有一套画布、节点和工作区交互。

## 画布导航

| 操作 | 鼠标 / 触控板 | 键盘修饰键 |
| --- | --- | --- |
| **平移画布** | 中键拖拽、`Alt` + 左键拖拽，或精确触控板双指滚动 | 左键拖拽时需要 `Alt` |
| **缩放画布** | 滚轮或捏合手势 | 滚轮缩放时使用 `Ctrl` |

触控板建议：

- 双指滚动用于平移视口
- 支持的设备上可直接捏合缩放
- 如果宿主环境不提供中键，也可以用 `Alt` + 左键拖拽作为平移回退手势

## 节点编辑

| 操作 | 快捷键 |
| --- | --- |
| **选择节点** | 左键单击 |
| **追加选择** | `Shift` + 左键单击 |
| **切换选择** | `Ctrl` + 左键单击 |
| **框选** | 在空白画布上左键拖拽 |
| **删除选择** | `Delete` |
| **复制选择** | `Ctrl + C` |
| **粘贴选择** | `Ctrl + V` |
| **撤销** | `Ctrl + Z` |
| **重做** | `Ctrl + Y` 或 `Ctrl + Shift + Z` |

## 工作区命令

| 操作 | 快捷键 |
| --- | --- |
| **保存文档** | `Ctrl + S` |
| **加载文档** | `Ctrl + O` |

## 宿主接入建议

如果你把 `NodeCanvas` 当成独立表面嵌入，并且希望快捷键完全由宿主管理，可以关闭默认绑定：

```xml
<avalonia:NodeCanvas EnableDefaultCommandShortcuts="False" />
```

这样做之后，最好在宿主文档里明确说明 `Ctrl+Z`、`Ctrl+S`、剪贴板等命令到底由谁接管，避免用户猜测。
这也是 canonical runtime/session 路线上自定义 UI 宿主采用的同一种快捷键所有权模式。
