# 采用者分诊清单

提交公开 beta 回报时，统一使用一组证据字段：

- `version`（包版本或公开 tag）
- `route`（尝试的路线或样例）
- `proof` marker（命中的 proof 标记）
- `support bundle`（support bundle 路径、打包文件或命令输出）

同一套字段应在：

- `adoption_feedback.yml` issue 模板
- Bug 报告模板
- 预发布 notes 的证据复核

确保所有回流内容保持一致后再进入发布评审。
