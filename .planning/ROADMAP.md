## Overview

AsterGraph now ships a public-alpha-ready baseline with a clean operational story: the four-package SDK boundary is intact, the canonical factory/session path is the recommended host route, `HostSample` proves minimal consumption, the demo showcases plugins and automation on that same path, and both branch CI plus prerelease validation run successfully on GitHub-hosted runners.

There is no active milestone. v1.9 is archived, and the next step is to start a fresh requirements pass for whatever comes after public launch-gate closure.
