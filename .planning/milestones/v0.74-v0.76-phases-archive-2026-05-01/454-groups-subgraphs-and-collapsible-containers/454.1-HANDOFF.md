## Contract Added Or Tightened

- `GraphNodeGroup` and `GraphEditorNodeGroupSnapshot` now expose explicit container semantics:
  - all node groups are single-scope editor containers;
  - `ProjectsMemberNodes` is `false` when the group is collapsed;
  - new core model properties are `JsonIgnore` so existing persisted documents stay compatible.
- `GraphEditorHierarchyStateSnapshot` now includes per-connection hierarchy projection.
- Collapsed group edge semantics are explicit:
  - a connection whose endpoints are both hidden by the same collapsed group is internal and not visible in the active scope;
  - a connection with exactly one hidden endpoint, or endpoints hidden by different collapsed groups, crosses a collapsed group boundary and remains visible for boundary-edge presentation;
  - ordinary connections with no collapsed endpoint remain visible.

## Residual Risk

- 454.1 only adds runtime/query contracts. Avalonia chrome, hit testing, selection projection, and visual boundary-edge rendering still need to consume these semantics in 454.2.
