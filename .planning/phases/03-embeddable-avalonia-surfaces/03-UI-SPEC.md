# Phase 03 — UI Design Contract

> Visual and interaction contract for frontend phases. Generated by gsd-ui-researcher, verified by gsd-ui-checker.

---

## Design System

| Property | Value |
|----------|-------|
| Tool | none — Avalonia-only manual contract sourced from `03-CONTEXT.md`, `03-RESEARCH.md`, and the existing `AsterGraphTheme` |
| Preset | not applicable |
| Component library | Avalonia built-in controls plus AsterGraph public surfaces: full shell, standalone canvas, standalone inspector, standalone mini map, and stock context-menu presenter |
| Icon library | none — the stock menu uses built-in text glyph chips from `GraphContextMenuPresenter` |
| Font | Inter for demo/default-shell validation; standalone library surfaces inherit the host font when a host does not set one explicitly |

---

## Visual Hierarchy

| Surface | Primary Focal Point | Secondary Focal Point | Accessibility Rule |
|---------|---------------------|-----------------------|--------------------|
| Full shell | The graph canvas viewport and current node scene | Inspector title plus current selection summary | Menu glyph chips and compact badges always appear next to text labels, never as unlabeled icon-only actions |
| Standalone canvas | The node graph scene itself | Selection rectangle, selected node border, and pending connection emphasis | Built-in context-menu items must keep text labels even when glyph chips are shown |
| Standalone inspector | Current node title and selection caption | Section headings such as `Connections` and `Parameters` | Field editors keep visible labels; no icon-only controls |
| Standalone mini map | The viewport rectangle inside the map | Node extents and graph bounds | No icon-only affordances; viewport navigation remains pointer-driven |

## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Micro gaps between caption lines, badge offsets, and dense inline metadata |
| sm | 8px | Canvas frame inset, compact card spacing, menu item vertical rhythm |
| md | 16px | Default control spacing, template card padding, and baseline standalone-surface gaps |
| lg | 24px | Standalone inspector padding, shell card padding, and section padding |
| xl | 32px | Layout gaps between independently embedded surfaces in host compositions |
| 2xl | 48px | Large empty-state breathing room and major shell section breaks |
| 3xl | 64px | Outer host padding around full-shell compositions |

Exceptions: none

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `#09111A / #0C1825 / #0A121C` | Full-shell gradient background and outer surface backdrop |
| Secondary (30%) | `#111C29` | Cards, inspector shell, library shell, status chrome, and stock context-menu body |
| Accent (10%) | `#7FE7D7` | Selected node borders, pending/compatible connection emphasis, mini-map viewport frame, and high-signal inspector/status emphasis only |
| Destructive | `#FF9B9B` | Destructive actions and inline validation errors only |

Accent reserved for: selected node outlines, pending or compatible connection emphasis, mini-map viewport framing, inspector category emphasis, and host-signaled ready or preview badges. Do not use accent as the default button fill, panel background, or generic hover color.

## Copywriting Contract

| Element | Copy |
|---------|------|
| Primary CTA | Add Node |
| Empty state heading | Select a Node |
| Empty state body | Select a node on the canvas to inspect its ports, connections, and editable parameters. If the session is empty, add a node first. |
| Error state | Action failed. Retry the action, or surface the editor session status message in host chrome. |
| Destructive confirmation | `Delete Selected`: Delete the selected nodes and connections? Undo is available only while the session stays open. `Clear Fragment`: Clear the saved fragment workspace? This removes the stored fragment snapshot. `Delete Template`: Delete the selected template? This removes it from the fragment library for this host. |

## Interaction Contract

| Surface | Contract |
|---------|----------|
| Shared session binding | All Phase 3 surfaces bind directly to the same `GraphEditorViewModel` and Phase 2 session-backed state. Do not introduce a second Avalonia-only state layer. |
| Full shell | `GraphEditorView` remains the convenience composition root and keeps header, library, canvas, inspector, and status chrome available under one surface. It preserves the current theme language and remains the easiest stock host path for `EMBD-01`. |
| Optional chrome | Header, library, and status chrome become omittable through composition, but they remain shell-only presenters in Phase 3. Do not promote them into separate first-class public controls yet. |
| Standalone canvas | The canvas surface is public and open-box usable. Default behaviors stay on: node drag, port activation, connection creation and cancellation, selection, marquee selection, wheel zoom, pan, right-click menu, and pointer-driven focus. Host-facing flags must expose `EnableDefaultContextMenu=true` and `EnableDefaultCommandShortcuts=true` by default. Command shortcuts covered by the default canvas contract are `Ctrl+S`, `Ctrl+O`, `Ctrl+Z`, `Ctrl+Shift+Z`, `Ctrl+Y`, `Ctrl+C`, `Ctrl+V`, and `Delete`. `Escape` must always cancel a pending connection as core graph interaction, even when command shortcuts are otherwise disabled. |
| Standalone inspector | The standalone inspector binds to the same session and stays pure-inspector only. It shows title, category, selection caption, description, `Connections`, `Inputs`, `Outputs`, `Upstream`, `Downstream`, `Parameters`, and inline validation. It must exclude workspace controls, fragment or template controls, shortcut-help blocks, and the mini map. |
| Standalone mini map | The mini map remains a narrow overview-and-viewport-navigation surface. It renders node extents plus the current viewport rectangle, supports click or drag recentering, stays non-focusable, and omits menus, status text, toolbar actions, or shell badges. |
| Default menu surface | The stock Avalonia context-menu presenter becomes a public default surface. Preserve the current stock menu treatment: background `#111C29`, border `#22354A`, hover `#193045`, foreground `#F4FBFF`, `220px` minimum width, `16x8` item padding, `10px` menu corner radius, `8px` item corner radius, and `16px` icon chip. Phase 3 stops here; do not introduce alternative presenter interfaces. |
| Focus and input ownership | Canvas shortcuts only fire while the canvas is focused. `TextBox` and `ComboBox` editors inside the inspector suppress graph shortcuts. The mini map never takes keyboard focus. Shortcut-help copy remains full-shell only and must not ship inside the standalone inspector. |
